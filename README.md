# ⚡ FitLab Pro

A modern fitness workout tracker built with pure HTML, CSS, and JavaScript.

FitLab Pro helps you track workouts, monitor progress, manage training programs, and record body statistics — all in one place.

## ✨ Features

### 🏋 Workout Tracking
- Log workouts
- Add exercises
- Track sets, reps, weight, and RPE
- Save workout notes
- View workout history

### 📈 Progress Tracking
- Track strength progression
- View:
  - Max weight
  - Volume
  - Estimated 1RM
  - Max reps
  - Total sets

### 📋 Training Programs
- Create custom programs
- Follow workout splits
- Manage training days

### ⚖ Body Statistics
Track:
- Bodyweight
- Waist measurements
- Arm measurements
- Progress over time

### 🧮 Fitness Tools
Includes:
- 1RM calculator
- Plate calculator
- Warm-up calculator
- Rest timer

### 🗂 Data Management
- Export your data
- Import saved data
- Works directly in your browser
